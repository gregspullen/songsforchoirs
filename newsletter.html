This Month at Songs for Choirs

Hello Choristers,

Welcome to songsforchoirs.This is the first chance I've had to write a column.

The last few months have been taken up with coding, testing and more coding and only thanks to Lyra has the site reached the point where I can begin to relax and allow the site to do it's thing. We believe that we have created a unique publishing system to give you the best possible experience. Now all we have to do is give you some music that is rewarding and fun to sing. I think that's easier than writing a single page of code. Lyra would disagree.

In between coding sessions we have managed to write a handful of new works. Normally we would expect to produce one or two pieces each month but as this is our start-up there are a few extras.
